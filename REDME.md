**🔗 Project Overview**
       project is based on the Bank Marketing Machine Learning problem.

**👉 Goal:**
   - Predict whether a customer will subscribe to a term deposit (Yes/No) based on their data.

   - This is a classification problem in Machine Learning.

   - A bank runs marketing campaigns (phone calls)

   - Not all customers say “YES”

   - Identify which customers are more likely to subscribe


**1. Data Collection**
           Dataset contains customer details like:

  - Age

  - Job

  - Balance

  - Loan

  - Contact type

**2. Data Preprocessing**

  - Handling missing values

  - Encoding categorical data

  - Feature scaling

  - Removing outliers

👉 This step is very important for model accuracy

**3. Exploratory Data Analysis (EDA)**
  Used graphs like:

  - Bar charts

  - Box plots

  - Heatmaps

  **📌 Example insights:**

    - Customers without loans → more likely to subscribe
 
    - Longer call duration → higher success rate

**4. Model Building**
   Common models used in such projects:

  - Logistic Regression

  - Decision Tree

  - Random Forest

  - SVM

  - XGBoost


**5. Model Evaluation**

  - Accuracy

  - Precision

  - Recall

  - F1-score


**6. Conclusion**

  - Model helps predict customer behavior

  - Helps bank improve marketing strategy

  - Saves cost & increases conversion
